# Personal Portfolio

A modern, responsive personal portfolio built with Next.js, React, and Tailwind CSS. The portfolio showcases projects, detailed case studies, and technical skills using a clean, accessible, and dynamic interface.

## 🚀 Features

- **Next.js & App Router**: Fast, server-rendered React application built on the latest Next.js 16.
- **Tailwind CSS**: Utility-first styling for rapid, responsive UI development.
- **Dynamic Custom Theming**: Includes a custom theme changer component (powered by `next-themes`) allowing users to dynamically switch between multiple aesthetic presets (Default, Dark, Green, etc.).
- **Detailed Case Studies**: Beautifully designed project pages equipped with responsive, sticky sidebars to comfortably navigate through comprehensive design rationale and technical implementation details.
- **Fully Responsive**: Carefully crafted layouts providing a seamless user experience across mobile, tablet, and desktop viewports.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **UI & Styling**: React 19, [Tailwind CSS v4](https://tailwindcss.com/)
- **Theming**: `next-themes`

## 📦 Getting Started

To run this project locally, clone the repository and fire up the development server:

1. **Install dependencies:**
   ```bash
   npm install
   ```
   *(or `yarn install`, `pnpm install`, `bun install`)*

2. **Start the development server:**
   ```bash
   npm run dev
   ```
   *(or `yarn dev`, `pnpm dev`, `bun dev`)*

3. **Open the application:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser to see the portfolio.

## 📂 Project Structure

- `app/`: Contains the Next.js App Router pages and layouts, including specific case studies (e.g., `the-allex`).
- `components/`: Reusable React UI components like navigation headers, case study sidebars, and the theme switcher.
- `public/`: Static assets such as images, fonts, and icons.
